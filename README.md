1.

var → Old way to declare variable. It can be redeclared and updated.

let → Modern way. It can be updated but not redeclared.

const → Value cannot be changed after declaration.



2.
The spread operator is used to copy or combine arrays/objects.


3.
map() → Creates a new array after changing data.

filter() → Creates a new array with selected items.

forEach() → Loops through array but does not return a new array.


4.
Arrow function is a short way to write a function.


5.
Template literals are used to write strings with variables easily using backticks ( ).
